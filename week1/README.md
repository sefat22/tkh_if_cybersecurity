# Linux Foundations & FHS

## Overview
This week focused on navigating the Linux Filesystem Hierarchy Standard (FHS) and understanding the "hidden" areas of an Ubuntu system.

## Tasks Completed
* **Scavenger Hunt:** Located hidden tokens in system directories like `/etc` and `/var`.
* **Environment Setup:** Configured an SSH bridge between a headless VM and GitHub.
* **Automation:** Created custom bash aliases for system management.

## Key Learning
* Navigate a headless Linux filesystem using absolute and relative paths without relying on a Graphical User Interface (GUI).
* Understand and explain the Filesystem Hierarchy Standard (FHS), identifying the purpose of /, /home, /etc, /bin, and /var/log.
* Execute core terminal commands (pwd, ls, cd, mkdir, touch, cat) utilizing the correct command grammar (Command $\rightarrow$ Flag $\rightarrow$ Argument).
* NSA/NICE Framework - OM-ADM-001 (System Administration).
